## 快速使用

::: code-group

<!--@include: ./repository.md-->

```bash [安装依赖]
  # Enter project
  cd Antd-Template-Vue

  # Install denpendencies
  pnpm install

```

```bash [本地启动]
  # Start develop server
  pnpm dev

```

```bash [本地构建]
  # Run build create dist
  pnpm build

  # Start server and preview
  pnpm preview

```

:::
